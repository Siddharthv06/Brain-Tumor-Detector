# рџ“– About

BrainTumorAI is an AI-powered medical imaging application that detects and classifies brain tumors from MRI scans using a **Custom Convolutional Neural Network (CNN)** developed entirely from scratch.

The application provides an intuitive Flask-based web interface where users can upload MRI scans and receive instant predictions with confidence scores. The complete machine learning pipeline includes image preprocessing, CNN training, evaluation, and real-time inference.

The model classifies MRI images into four categories:

- рџ§  Glioma
- рџ§  Meningioma
